# Movie Library Management System

**D201 Advanced Programming — Assessment 1**

A WPF desktop application built in C# that manages a movie library for a small film club.

## Features
- Add, search and sort movies
- Borrow and return movies with a waiting queue
- Export and import movie data as JSON

## Data Structures Used
- Linked List (movie collection)
- Hash Table (fast lookup by ID)
- Queue (waiting list - FIFO)

## Algorithms Used
- Linear Search (by title)
- Binary Search (by Movie ID)
- Bubble Sort (by title)
- Merge Sort (by release year)

## How to Run
1. Open `MovieLibrary.sln` in Visual Studio 2022
2. Press F5 to build and run

## Technologies
- C# / WPF / .NET 9
- xUnit for unit testing
